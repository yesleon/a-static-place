Why I use these apps

Ulysses

Ulysses is my main creative space. I use it to write anything in a project. That is, professional writings such as essays, books, website articles.

Things

Things is my personal to-do list app. I also use it with minor personal projects or projects I’m not managing.

Trello

Trello is my professional project management app. If a project I manage is too big for Things, I use Trello to manage it instead.

MindNode

MindNode is like Ulysses which is also a creative tool. I use MindNode to do brainstorming, with or without peers. I also use it to manage my personal financial accounts because its non-linear structure is more suitable than a word document or a spreadsheet in this case.

PDF Expert

PDF Expert is my PDF reader for academic essays and book chapters. I also use it for every other PDFs instead of the built-in Preview app.

Day One

Day One is my journaling app. I’m still figuring out if I should use it to journal because sometimes it distracts me to journal on a digital device. However if I’m going to journal on my Apple devices I’m going to use Day One to do that.

Google Chrome

When a website is not that compatible with Safari, I use Chrome instead. For example some Flash-powered webpages or web apps like Google Drive.

1Blocker

It’s the only ad blocker I use throughout all my Apple devices. It’s efficient and highly customisable. I feel more safe with 1Blocker as well because its business model is one-time purchase.

DaisyDisk

I use DaisyDisk to get a whole picture of how my disk is used and to decide which folders to delete.

TextMate

TextMate is the plain text editor I use. Unless it’s an RTF file I’m not going to use TextEdit for opening a text file. TextMate has syntax colouring and a command line opener. It also supports Markdown previewing. It opens property list files. It even has version control functions built in. Very useful for a programmer.

The Unarchiver

Sometimes the built-in Archive Utility won’t open special archive formats like 7z files. The Unarchiver can unarchive them, that’s all.

Final Cut Pro

My video editing software choice because I like the way it handles its library. The Magnetic Timeline is both intuitive and not distracting.
